# Phần mềm miễn phí

Đây là website giới thiệu những phần mềm miễn phí cho những nhu cầu cơ bản của dân văn phòng, học sinh, sinh viên, người dùng máy tính Windows phổ thông.

- Domain: phanmemmienphi.test (staging), phanmemmienphi.net (production)
- Đối tượng: dân văn phòng, học sinh, sinh viên, người dùng máy tính Windows phổ thông
- Website chỉ là nơi giới thiệu ngắn gọn và có link download (hoặc link tới trang chủ) của phần mềm.
- Ngôn ngữ: tiếng Việt
- Stack: static site
- Deploy: Cloudflare pages/worker hoặc Github pages

## Danh sách phần mềm:

- Văn phòng: ONLYOFFICE, Thunderbird, Foxit PDF Reader, Unikey
- Tiện ích: 7-zip, ShareX, CCleaner
- Media: VLC, Affinity, DaVinci Resolve, Audacity
- Mạng: Google Chrome, UltraViewer, Zalo
- Bảo mật: Avast

## Cấu trúc

Mỗi phần mềm sẽ có:
- Tên
- Mô tả ngắn
- Giới thiệu đầy đủ
- Ảnh đại diện (icon)
- Ảnh chụp màn hình
- Link trang chủ
- Link download trực tiếp (nếu có)

Mỗi phần mềm có 1 trang riêng để giới thiệu chi tiết.
